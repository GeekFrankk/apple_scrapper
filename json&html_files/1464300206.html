    Description

    This new patient mobile application makes it easy to get the care you need, every step (and click) of the way.


    It’s easy to manage your healthcare and that of your loved ones. All of the same features you currently access in the MyChart app are included.


    Some key features in the app allow you to:

    • Access to general information

    • Access visitor information

    • View health records

    • View test results

    • Integration with the Health app lets you share health data with your provider

    • Schedule appointments

    • Message your doctor’s office

    • See a provider live for non-emergency concerns


    To access all the features, you must first create an account.
