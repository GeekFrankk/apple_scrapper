    Description

    Dose Chex is designed to simplify the calculations required when administering drugs and fluids. Rather than performing drug calculations manually, Dose Chex performs them for you. It is fast and reliable, and won’t be affected by working a long shift!


    You can create different medications as required and save them for future calculations. A number of safety features highlight potential sources of error, such as using the incorrect units.

    What’s New

    Jul 1, 2023

    Version 2.2

    In response to user feedback we have added a method to enter the strength of medications directly whilst retain the ability to save medications for future use. You can select which method to use as your preferred default but also have instant access to both methods when required.
