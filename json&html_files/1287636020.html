    Description

    What’s New

    Version 2.0.0

    BifurcAID has been updated to increase compatibility with newer phone operating systems. Additionally proximal optimization technique has been added throughout the Left Main algorithm as well as several minor fixes. Now BifurcAID is accessible without an internet connection after initial sign-in.
