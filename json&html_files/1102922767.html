    Description

    An interactive program designed for cardiovascular health care professionals who want to learn the basics of coronary intravascular ultrasound.


    Welcome to Coronary IVUS Tutor app, an interactive program designed for cardiovascular health care professionals who want to learn the basics of coronary intravascular ultrasound.


    Foundational basics of coronary intravascular ultrasound

    • Simulate IVUS pullbacks and measurements using the interactive workflow

    • View the image interpretation guide and learn how to quickly identify common vessel morphologies

    • Test yourself with the knowledge check quiz
