    Description

    Echocompanion is a dedicated echocardiography tool providing normal and abnormal ranges for various echocardiographic parameters based on the latest available guideline. Also, echo calculator is available. In addition, educational material in the field of echocardiography is available including The Book of Echo and many educational videos.
