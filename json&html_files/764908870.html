    Description

    Insulin-Rx application is a convenient insulin dose calculator intended for use by healthcare professionals who prescribe basal-bolus insulin therapy. Patient weight and total daily insulin can be efficiently entered in by using sliders instead of typing in the values. The calculator then displays insulin units/kg/day, carbohydrate-to-insulin ratio, insulin sensitivity (correction factor), fixed meal bolus dose, and daily basal dose.


    A calculator that converts hemoglobin A1C to the estimated corresponding average glucose value is also included as well as a converter of centimeters to inches together with a body mass index calculator.
